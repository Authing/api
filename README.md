## Programmable API Server

```
node index.js
```

This will setup 2 apis:

- `GET` `/list`: full api list
- `GET` `/api/:name`: query specific api detail

## Command Line Usage

### Print List

```
yarn all
```

### Print Specific API

```
yarn print <API-NAME>
```

For example: 
```
yarn print user
```
